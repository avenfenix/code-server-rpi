# code-server en Raspberry PI

Intentando correr code-server en mi Raspberry Pi 4b

## Requisitos

* NodeJS
* npm
* Dependencias code-server

## Instalar

### Dependencias
```
sudo apt-get install -y \
  build-essential \
  pkg-config \
  python3
npm config set python python3
```
